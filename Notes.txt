        OOP's Principles : 
        ------------------
                1. Inheritance 
                2. polymorphism
                3. Encapsulation
                4. Abstraction


1. Inheritance: 
        ->inheriting properties from one to another class another class is called a Inheritance
                        or 
        ->occurring properties from one class to another class is also called as a Inheritance

 Note : ->To perform code Reusability
        ->To achieve method overRidding Inheritance is required 
        ->To achieve Abstraction we required Inheritance
        

        Types of Inheritance:
                1. Single Level Inheritance 
                        ->single subClass/child Class/ Derived class occurring properties from only one superClass/parent Class /base Class


                2. Multi Level Inheritance
                        ->subClass occurring properties from superClass interns superClass occurring from its superClass


                3. hybrid Inheritance
                        ->the combination single level , multi level or hierarchical is called hybrid Inheritance

                4. hierarchical Inheritance
                        ->multiple subClass's occurring properties from only one superClass 
                        
                5. multiple Inheritance 
                        ->single subClass occurring properties from multiple superClass's 




        2. polymorphism : 
                ->same method showing different behavior throughout its life cycle
                ->in python to polymorphism we required Inheritance and method overRiding 

overRiding : 
         ->developing a same method in subClass as in ur superClass but different implements in subClass


        note:-> whenever we want to add new features to existing system
        ->In the case of method overRidding we will get overridden implements
